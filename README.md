# PDF Rotate

Application Flutter pour visualiser et faire pivoter des fichiers PDF.

## Fonctionnalités

- Ouverture de fichiers PDF depuis le stockage local
- Navigation entre les pages (précédent/suivant)
- Rotation de l'affichage (0°, 90°, 180°, 270°)
- Zoom interactif (pinch-to-zoom)
- Interface en mode sombre

## Installation

```bash
flutter pub get
flutter run
```

## Dépendances

- `pdfx` - Rendu PDF
- `file_picker` - Sélection de fichiers

## Plateformes supportées

- Android
- iOS
- Web
- Windows
- macOS
- Linux
